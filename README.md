1️⃣ What is the difference between var, let, and const?
#Ans: var ফাংশন-স্কোপড, অর্থাৎ শুধুমাত্র ফাংশনের ভিতরে কার্যকর। let এবং const ব্লক-স্কোপড, অর্থাৎ শুধু {} ব্লকের ভিতরে কাজ করে। let পরিবর্তনযোগ্য, কিন্তু const একবার মান দেওয়া হলে পুনঃনির্ধারণ করা যায় না।

2️⃣ What is the spread operator (...)?
#Ans: এটি অ্যারে বা অবজেক্টকে আলাদা এলিমেন্ট হিসেবে ছড়িয়ে দেয়। উদাহরণ: [...arr1, ...arr2] দিয়ে দুইটি অ্যারে একত্র করা যায় বা {...obj1, ...obj2} দিয়ে অবজেক্ট মিশানো যায়।

3️⃣ What is the difference between map(), filter(), and forEach()?
#Ans: map() প্রতিটি এলিমেন্টে ফাংশন প্রয়োগ করে নতুন অ্যারে তৈরি করে। filter() শর্ত পূরণ করা এলিমেন্ট নিয়ে নতুন অ্যারে তৈরি করে। forEach() কেবল লুপ চালায় এবং কোনো নতুন অ্যারে ফেরত দেয় না।

4️⃣ What is an arrow function?
#Ans: সংক্ষিপ্ত ফাংশন লেখার সিনট্যাক্স => ব্যবহার করে। এটি সাধারণ ফাংশনের মত কাজ করে, তবে this এর মান লেক্সিকাল, অর্থাৎ বাইরের কনটেক্সট অনুযায়ী থাকে। উদাহরণ: const add = (a, b) => a + b;

5️⃣ What are template literals?
#Ans: ব্যাকটিক ` ব্যবহার করে লেখা স্ট্রিং, যেখানে ${variable} দিয়ে ভ্যারিয়েবল যোগ করা যায়। এছাড়াও মাল্টি-লাইন স্ট্রিং লেখা সম্ভব।
